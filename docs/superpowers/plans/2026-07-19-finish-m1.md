# Finish Milestone 1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans
> to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Close the four M1 DoD items — build the leaf app + shim, prove
`spumePimpleFoam` is byte-identical to stock `pimpleFoam` via a reusable
equivalence oracle, and document one upstream rebase dry-run.

**Architecture:** Commit the in-flight shim/leaf/scripts; a bitwise field+log
comparator (`tests/regression/`) drives both solvers on checked-in pimpleFoam
fixtures, gated in the nightly OpenFOAM CI; motorBike runs locally as the
headline. The comparator carries a `bitwise|reorder-tolerance` mode so M2 reuses
it as its regression gate.

**Tech Stack:** OpenFOAM v2606 (wmake), POSIX sh, Python 3, CMake/ctest,
GitHub Actions.

## Global Constraints

- OpenFOAM API floor: **v2606** (`OPENFOAM >= 2606`, ADR-0014).
- Vendored tree edited only via `patches/` (≤ **2,000 lines**, ADR-0001); never edit `vendor/` directly.
- SPUME reaches upstream only through `src/compat/` (invariant #2).
- Every new file: SPDX header + `Copyright (C) 2026 Joern Ihlenburg`.

- Never change numerics defaults or weaken tests; bitwise where achievable, never a tolerance to force a pass.
- No new dependencies without approval; no benchmark/counter claim without evidence (ADR-0013).
- Branch: `milestone1/finish`. Naming: SPUME/`spume`.

### Task 3: The equivalence comparator (`check_equivalence.py`) — TDD

**Files:**
- Create: `tests/regression/check_equivalence.py`
- Test: `tests/regression/test_check_equivalence.py`

**Interfaces:**
- Produces:
  - `strip_banner(text: str) -> str` — removes the leading `/*--…--*/` FoamFile banner block.
  - `fields_equal(a: bytes, b: bytes, mode: str) -> bool` — `mode in {"bitwise","reorder-tolerance"}`; M1 uses `"bitwise"` (banner-stripped byte compare). `"reorder-tolerance"` is a documented stub raising `NotImplementedError` until M2.
  - `residual_lines(log: str) -> list[str]` — the `Solving for …` lines, execution-time lines dropped.
  - CLI: `check_equivalence.py <spumeCaseDir> <stockCaseDir>` → exit 0 if all written time-dir field files are bitwise-equal and residual lines match, else non-zero with a diff summary.

- [ ] **Step 1: Write failing tests.**

```python
# tests/regression/test_check_equivalence.py
import check_equivalence as ce

BANNER = "/*---*\\\n| foo |\n\\*---*/\n"

def test_strip_banner_removes_header():
    assert ce.strip_banner(BANNER + "payload\n") == "payload\n"

def test_fields_equal_bitwise_true_when_only_banner_differs():
    a = (BANNER + "1.0 2.0 3.0\n").encode()
    b = (BANNER.replace("foo", "bar") + "1.0 2.0 3.0\n").encode()
    assert ce.fields_equal(a, b, "bitwise") is True

def test_fields_equal_bitwise_false_on_data_diff():
    a = (BANNER + "1.0\n").encode()
    b = (BANNER + "1.1\n").encode()
    assert ce.fields_equal(a, b, "bitwise") is False

def test_residual_lines_drops_exec_time():
    log = "Solving for Ux, Initial residual = 1e-3\nExecutionTime = 0.5 s\n"
    assert ce.residual_lines(log) == ["Solving for Ux, Initial residual = 1e-3"]

def test_reorder_tolerance_is_stub():
    import pytest
    with pytest.raises(NotImplementedError):
        ce.fields_equal(b"a", b"b", "reorder-tolerance")
```

- [ ] **Step 2: Run, verify fail.** `cd tests/regression && python3 -m pytest test_check_equivalence.py -v` → FAIL (module missing).

- [ ] **Step 3: Implement `check_equivalence.py`** with SPDX header, the four functions above, and a `main()` walking both case dirs' time directories. `strip_banner` regex: `^/\*-+.*?\*-+/\s*` DOTALL at start. `residual_lines`: keep lines containing `Solving for`, drop those containing `ExecutionTime`/`ClockTime`.

- [ ] **Step 4: Run, verify pass.** Same pytest command → PASS (5 tests).
